4 PCA

Letâ€™s create some dummy explanatory data. We will generate four predictors, with negative correlations between the first and second and between the third and fourth:

set.seed(3)
library(mvtnorm)
x.raw <- rmvnorm(50,mean=rep(0,4),
        sigma=matrix(c(1,-0.8,0,0,
                       -0.8,1,0,0,
                       0,0,1,-0.8,
                       0,0,-0.8,1),
                     nrow=4,byrow=TRUE))

# for the sake of simpler calculations later on, let's center x already now:
x <- apply(x.raw,2,function(x) x - mean(x))

When we now try to calculate the principal components, we find;

a <- prcomp(x)
summary(a)
## Importance of components:
##                           PC1    PC2     PC3     PC4
## Standard deviation     1.5386 1.2615 0.41082 0.38645
## Proportion of Variance 0.5535 0.3721 0.03946 0.03492
## Cumulative Proportion  0.5535 0.9256 0.96508 1.00000

Hence, the first two axes together already explain over 90% of the data. We could thus decide in a predictive model to only use these two axes.

newx <- predict(a)
# because we centered x beforehand, this is equal to:
# newx <- x %*% a$rotation

y <- apply(x,1,sum) + rnorm(nrow(x),0,0.5)
d <- as.data.frame(cbind(y=y,newx))
library(MuMIn)
## Warning: package 'MuMIn' was built under R version 4.2.3
